# 房间列表

|  键   | 值  | 说明 |
|  ----  | ----  | ---- |
| mesg  | room_list ||
| room_list  | [1,2,3,4] | 房间id数组|

# 添加房间

|  键   | 值  | 说明 |
|  ----  | ----  | ---- |
| mesg  | add_room ||
| type  | amazons |amazons或surakarta|
| args | [10, 10] | 棋盘大小 |
| time | 900 | 时间 (s) |
| order | 1 | 1为先手，2为后手 |

# 加入房间

|  键   | 值  | 说明 |
|  ----  | ----  | ---- |
| mesg  | join ||
| room_id  | 2 | 房间号 |
| order | 1 | 1为先手，2为后手 |

## 添加和加入房间的返回值

|  键   | 值  | 说明 |
|  ----  | ----  | ---- |
| playerId | 'A' | 玩家ID |

# 开始

|  键   | 值  | 说明 |
|  ----  | ----  | ---- |
| mesg  | start ||


## 返回值

|  键   | 值  | 说明 |
|  ----  | ----  | ---- |
| state  | ready | 如果对方没有开始则返回ready,否则返回start |
